# Sprintern Clone

A clone for Sprintern, a platform for giving practical hand-on internship experience.

## Project setup steps

1. Clone this project 
2. Change Directory [`cd s_clone`]
3. Install Dependencies using npm [`npm i`]
4. Run Local Server [`npm run dev`]

## TechStack

Used React, HTML5, TailwindCSS, and IndexedDB.
